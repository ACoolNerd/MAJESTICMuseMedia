# MAJESTICMuseMedia.ai

**MAJESTIC Muse Podcast by Marchette**  
*Where faith, purpose, and identity find their voice.*

A mobile-first media intelligence and production operating system for episodes, guests, livestreams, CapCut handoffs, approvals, rights, distribution, analytics, forms, community, and AI-assisted repurposing.

## Foundation status

This branch contains the production-ready application shell and mock operating data. External services remain behind adapters and show **Connection Required** until credentials are configured.

## Start locally

```bash
cp .env.example .env.local
npm install
npm run dev
```

## Quality checks

```bash
npm run typecheck
npm run lint
npm run test
npm run build
```
